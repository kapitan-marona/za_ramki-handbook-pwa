# 8. Key Flows

## 8.1 Work Flow (Staff)

Planner → Task → Checklist → Done

## 8.2 Knowledge Flow

Articles → Read → Apply in task

## 8.3 Admin Flow

Admin → Content → Edit → Publish

## 8.4 Project Setup Flow

Admin → Projects → Create → Use in tasks

---

# 9. Structural Issues (from map)

* navigation and data mixed (Planner left panel)
* admin is embedded, not separated
* editors are duplicated instead of unified
* lists share pattern but not system

---

# 10. Final Structure Insight

System can be simplified into:

1. Work (Planner)
2. Knowledge (Articles/Templates)
3. Execution (Checklists)
4. Management (Admin)

This is the real product architecture.
